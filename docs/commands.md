# コマンド仕様

全てスラッシュコマンド。プレフィックスコマンドは使用しない。登録は `pnpm run deploy-commands` で実行。`DEPLOY_GUILD_ID` 設定時はギルドコマンド（即時反映）、未設定時はグローバルコマンド。

## /join

- VC に参加し、コマンド実行チャンネルを読み上げ対象に設定
- ユーザーが VC に参加している必要あり
- Bot が別 VC に接続中 → エラー (「現在 🔊VC-1 で使用中です。/leave で退出してから再度 /join してください」)
- Bot が同じ VC に接続中 → 読み上げ対象チャンネルを切り替え (「読み上げチャンネルを #チャンネル名 に変更しました」)
- サーバー設定で挨拶 ON → 「接続しました」を読み上げ

## /leave

- VC から退出。即座にキュークリア、再生中断、退出
- 通常ユーザー: Bot と同じ VC に参加必要
- `ManageGuild` 権限 or 管理ロール (`adminRoleId`): VC 未参加でも実行可能

## /voice

- ユーザー単位の音声設定管理 (userId キー、全サーバー共通)
- サブコマンドなし
- 現在設定を Embed 表示 → View (セレクトメニュー・ボタン・モーダル) で変更
- 話者一覧: セレクトメニュー (25件超はボタンでページ切替)
- 速度・ピッチ: ボタン → Modal で数値入力
- 話者一覧は VOICEVOX `GET /speakers` から動的取得、Bot 起動時にメモリキャッシュ
- 速度: 0.5〜2.0、ピッチ: -0.15〜0.15
- 設定変更はどのサーバーでも可能。読み上げ適用は PREMIUM サーバーのみ

## /settings

- サーバー単位の設定。`ManageGuild` 権限 or 管理ロールのみ実行可能
- カテゴリ切り替え方式 UI (セレクトメニューでカテゴリ選択)

### 読み上げ設定カテゴリ
- 最大文字数 (デフォルト50。FREE: 変更不可・上限50、PREMIUM: 上限200)
- 名前読み上げ ON/OFF (デフォルト OFF)
- さん付け ON/OFF
- ローマ字読み ON/OFF

### 通知設定カテゴリ
- 入退室通知 ON/OFF (参加/退出/移動。さん付けはサーバー設定連動)
- Bot 入室時の挨拶 ON/OFF

### フィルタ設定カテゴリ
- カスタム絵文字の扱い (名前読み上げ or 除去)
- 読み上げ対象種別 (テキストのみ / +スタンプ / +スタンプ+添付ファイル)

### 接続設定カテゴリ
- 自動接続 ON/OFF
- デフォルト読み上げチャンネル (未設定→自動接続無効扱い)
- デフォルト話者

### 権限設定カテゴリ
- 管理ロール (`ManageGuild` 権限者のみ設定可能)
- サーバー辞書追加権限 (「全ユーザー」or「管理者 or 指定ロールのみ」。デフォルト: 後者)

## /dictionary

- グローバル辞書とサーバー辞書の2層。タブ切り替え UI

### サーバー辞書一覧
- 10件/ページ。管理者 or 指定ロールのみ削除ボタン表示。削除前確認あり

### サーバー辞書追加
- Modal で単語+読み入力。追加権限はサーバー設定に従う
- 重複チェックあり。上限: FREE 10件、PREMIUM 100件

### グローバル辞書一覧
- ページネーション付き。閲覧のみ

### グローバル辞書申請
- Modal で単語・読み・理由を入力。全ユーザー申請可能
- `GLOBAL_DICT_NOTIFICATION_CHANNEL_ID` のチャンネルに承認/却下ボタン付き Embed を送信

### 申請確認 (Bot 管理者のみ)
- 未承認申請一覧表示。承認/却下ボタン
- Bot 管理者 = `BOT_ADMIN_USER_IDS`

### 辞書バリデーション
- 単語: 最大50文字、空白のみ・空文字禁止
- 読み: 最大100文字、空白のみ・空文字禁止、ひらがな・カタカナのみ

## /help

- コマンド一覧と概要を Embed で表示 (通常表示、Ephemeral ではない)
- HP の URL へ誘導 (HP 未公開時は URL 省略)
